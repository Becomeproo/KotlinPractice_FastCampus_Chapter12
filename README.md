# KotlinPractice_FastCampus_Chapter12-도서 리뷰 앱
### 해당 챕터를 통해 배운 것
* RecyclerView 사용
* View Binding 사용
* Retrofit 사용 (API 호출)
* Glide 사용 (이미지 로딩)
* Android Room 사용
* Open API 사용

### 도서 리뷰 앱
* 인터파크 Open API를 통해 베스트셀러 정보를 가져와서 화면에 출력
* 인터파크 Open API를 통해 검색어에 대항하는 책 목록을 가져옴
* Local DB를 이용하여 검색 기록을 저장하고 삭제
* Local DB를 이용하여 개인 리뷰 저장
